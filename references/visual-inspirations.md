# Visual Inspirations

These GitHub projects are useful references when generating a more game-like `skill-tree.html`.

## References

- [beautiful-skill-tree](https://github.com/andrico1234/beautiful-skill-tree): useful for seeing how a skill-tree UI can feel like a real progression map instead of a checklist.
- [RPG Maker MV Skill Trees System](https://github.com/SomeFire/RPG-Maker-MV-Skill-Trees-System): useful for node unlocking, branch structure, and RPG-style progression.
- [rpg-companion-sillytavern](https://github.com/anotherusername/rpg-companion-sillytavern): useful for status panels, character-like widgets, and progress-meter styling.
- [tabler](https://github.com/tabler/tabler): useful for clean dashboard hierarchy, spacing, card rhythm, and responsive panel layout.
- [skillmap](https://github.com/search?q=skillmap&type=repositories): useful as a general reminder to make prerequisite relationships explicit in the layout.

## What To Borrow

- Dark UI base with controlled contrast.
- Strong section hierarchy.
- Neon, gold, or blue accent highlights.
- Visible branches, node states, and progress meters.
- Header-level XP bar and compact status block.
- A map-like feeling, not a flat checklist.

## What To Avoid

- Generic gray admin cards.
- Flat lists with no branch identity.
- Excessive animation or noisy decoration.
- Any design that hides progression behind plain text only.
