# Adapters

This directory contains platform-specific adapters for the platform-neutral Learning Workflow Designer package format.

Available adapters:

| Adapter | Status | Purpose |
|---|---|---|
| `claude-code/` | Initial usable adapter | Install as a Claude Code skill for local codebase learning and project-lab generation. |

Adapter rule: keep platform-specific installation and prompt details here, while the core package format remains in `docs/learning-package-spec.md`.
